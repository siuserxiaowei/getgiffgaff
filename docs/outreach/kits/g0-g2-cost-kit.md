# G0/G2 现金支出计算器编辑素材包

适用对象：维护英国 SIM、giffgaff 选卡或购买成本教程的编辑。本素材用于解释公开算法和判断边界，不提供报价、库存、余额、购买推荐或外链结果。

## 唯一事实源与落地页

- 唯一事实源：[外链与合作内容唯一事实源](../content-source-of-truth.md)，定位以其中第 2、4、5 节为准。
- 工具：<https://getgiffgaff.com/tools/g0-g2-total-cost/>
- G0/G2 解释页：<https://getgiffgaff.com/answers/>

工具只计算用户自己输入的一条购买路径。它没有预置商品价格，也没有读取实时商品、订单、库存或账号数据。

## 公开现金公式

```text
现金支出 = 卡价 + 运费 + 首次充值 + 预计额外使用支出
```

- 四个现金输入均按人民币记录，并作为本次现金支出相加。
- 可用余额按英镑单独记录，不加入人民币现金合计，也不按未知汇率折算或抵扣。
- 每次只运行一条路径。比较 G0 与 G2 时，使用相同的预计使用范围分别运行两次，再由编辑或读者人工比较。
- 输出只代表“按本次输入计算”的算术结果，不是报价、实际支付额或总体拥有成本。
- 工具没有时间、维护小时、机会成本、汇率、手续费、退款、售后或未来支出的自动模型。

## 实体与判断边界

G0/G2 是 getgiffgaff 用于销售和教程的本站库存/状态分类，不是 giffgaff 官方 SKU、套餐、账号等级或合规结论。

计算器不能判断：

- 账号控制权、注册责任、恢复能力或 SIM swap 风险；
- 激活、普通短信、OTP 验证码或特定平台兼容性；
- 当前库存、实付价格、余额、发货安排或支付结果；
- 某批卡、交易方式、账号安排或销售流程是否合规；
- G0 或 G2 哪个更适合某人、更可靠或更值得购买。
