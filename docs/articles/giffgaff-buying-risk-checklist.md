# 购买 giffgaff / GG 卡前的风险检查清单

更新日期：2026-07-01

## 先说结论

购买 giffgaff / GG 卡前，先确认自己要的是长期英国号码，而不是一次性验证码工具。GG 卡可以作为 +44 号码、低频短信和账号备用方案，但它不能保证所有平台验证码必到，也需要后续保号和账号维护。

下单前最该问清楚的是：买 G0 还是 G2、当前是否有货、G2 余额范围、发货方式、售后排查需要哪些截图，以及你要绑定的平台是否接受英国号码。

社区里常见的“验证码必到”“低成本保号”“WiFi Calling abroad 省钱”都先当作待核查说法，不要当购买决策里的确定承诺。官方页显示 WiFi Calling abroad 不支持；VoLTE 可在英国和海外工作，但它不等于验证码和漫游费用问题都被解决。

## 适合谁看

这篇适合第一次购买 giffgaff 的中文用户，也适合正在比较 G0 新卡、G2 有余额卡、实体 SIM 和 eSIM 的人。

如果你只是想马上注册某一个平台，这篇尤其要看。因为验证码能否送达不只取决于 SIM，还取决于目标平台、账号资料、IP、设备和请求频率。

## 操作清单

下单前按这个清单确认：

1. 明确用途：长期保留英国号码、低频收短信、账号备用，还是短期注册。
2. 选择卡种：会自己激活和充值可以看 G0；第一次买、急用或担心支付失败，优先看 G2。
3. 确认库存：不要只看旧截图，付款前问当前是否有货。
4. 确认余额：购买 G2 时，问清当前余额范围，不要默认每张都一样。
5. 确认发货：从哪里发、用什么快递、是否能加急、丢件怎么处理。
6. 确认售后：无信号、无法登录、收不到短信时，需要提供哪些截图和信息。
7. 确认维护成本：是否愿意记录账号、定期保号、检查余额和漫游资费。
8. 确认边界：不要把“能收短信”理解成“所有验证码都保证能到”。
9. 核查传言：低成本保号、WiFi Calling 省钱、长期不用也不会停等说法，都要回到官方页面确认。

收到卡后，先保存订单、卡面、账号和号码信息。不要第一时间连续测试多个平台验证码，先确认信号、账号、余额和普通短信。

## 常见踩坑

第一坑是只比价格。便宜不等于适合你，G0 和 G2 的操作难度、付款环节和到手状态不同。

第二坑是没有问余额和库存。G2 有余额卡的余额范围和库存可能变化，付款前确认比付款后争论更有效。

第三坑是把售后当成“保证所有平台成功”。售后通常能帮你排查卡状态、信号、账号和基础短信，但平台风控不完全由卖家或 SIM 决定。

第四坑是买回来不维护。giffgaff 号码长期不用可能被视为 inactive，后续还要自己做保号记录。

第五坑是用社区“20 年保号”代替官方规则。购买前可以参考用户经验，但文章、下单和售后判断都应按官方 6 months inactive 规则来。
