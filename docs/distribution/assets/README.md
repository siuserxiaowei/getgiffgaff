## 发布前仍要做的事

1. 重新核对文章中的当天官方来源；图片不替代官方来源截图。
2. 在正文首次出现 getgiffgaff 或链接前，加入文章包中规定的独立第三方身份披露；出现咨询/销售语境时也加入商业披露。
3. 每篇只保留一个对应深层页链接。平台不支持外链时，不在评论区反复补链或诱导私信。
4. 两套小红书图文现有各 6 张原创图卡。发布当天至少另补 1 张当天官方来源截图（不含账号、号码、验证码、订单或支付信息），让规则事实可复核；不要用 AI 图卡替代官方来源。

SVG 是可编辑源文件；JPG 是直接上传版；PNG 是无损上传版。选择其一上传即可，不要同一张重复上传。六图源文件使用 `*-xiaohongshu-set.svg`；已经按发布顺序拆分为六张 JPG/PNG。
